PRIMER ITERACION
Página de Repartos: 

Al ingresar se ve la lista de repartos previamente cargados.
Con el botón “Nuevo Reparto” se habilitan los campos de carga o haciendo click en la grilla seleccionando un pedido anterior.
Al presionar “Guardar Reparto” se guardan los datos cargados y se agrega el registro a la grilla.
En la grilla se ´puede seleccionar el check “Eliminar” para eliminar un reparto.
Al clickear sobre un pedido se puede ver la localización en el mapa.
 

SEGUNDA ITERACION
Desde el botón “Zonas de Reparto” se accede a la página para crear, modificar o eliminar zonas de reparto.
Si se están agregando datos de pedidos y se presiona el botón “Zonas de Reparto”, se debe avisar que se van a perder los datos cargados y confirmar si desea ir a la carga de Zonas.




Zonas de Reparto:

Desde esta página pueden crearse, modificarse o eliminarse zonas de reparto.
Para modificar un Zona existente se clickea sobre una de la grilla, y automáticamente se completan los datos en las casillas de ingres de datos, y se ve en el mapa el polígono definido para la zona. Se puede ajustar los datos de “Zona”, “Transporte” y los puntos del polígono sobre el mapa.
Para dar de alta se presiona el botón “Nueva Zona”, y se permite ingresar los datos de “Nro de Zona”, “Zona”, “Transporte”, y definir los puntos del polígono de la zona.
Con “Guardar Zona”, se guardan los datos ingresados, para altas o modificaciones.
Para eliminar una zona se puede seleccionar una en la grilla y presionar el botón “Eliminar Zona”.
Para volver a la carga de Repartos, se puede presionar el botón “Repartos”.
Si se están agregando datos de Zonas y se presiona el botón “Repartos”, se debe avisar que se van a perder los datos cargados y confirmar si desea ir a la carga de Repartos.

 
Hay que modificar la carga de Repartos para que cuando se ingrese la dirección la zona se selecciona automáticamente.
